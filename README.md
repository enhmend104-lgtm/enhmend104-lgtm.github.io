# 핐사관리시스템 (Academic Management System) · B3

대학 강의 관리 및 수강신청을 위한 **웹 애플리케이션**입니다.
B3팀의 «학사관리시스템» 기획서를 바탕으로, AI 도구(Claude)를 활용해 제작했습니다.
🔗 **배포 주소:** `https://<your-username>.github.io`

## 주요 기능

- **권한 분리 로그인** — 학생 / 교수 / 관리자 계정이 각각 다른 기능을 사용합니다.
- **강의 검색** — 과목명·강의코드·교수명 검색, 요일 필터, 신청 가능 강의만 보기.
- **수강신청 / 취소** — 정원 만감, 시간표 충돌, 최대 학점(21학점) 자동 검사.
- **주간 시간표** — 신청한 강의를 요일×교시그리드로 확인.
- **강의 등록 / 수정** (교수) — 강의실 수용 인원·강의실 중복·교수 시간 충돌 검사.
- **계정 / 강의실 관리** (관리자).

모든 데이ᄰ는 브라우저의 `localStorage`에 저장되어 섈로고캭해도 유지됩니다.
(우측 하단 «데이터 초기화»로 초기 상태로 됌돌릴 수 있습니다.)

## GitHub Pages 배포 방법

1. GitHub에서 새 저장소를 만듭니다. **이름은 `<your-username>.github.io`** 로 하세요.
   (예: 아이디가 `moonjuseon`이면 저장소 이름은 `moonjuseon.github.io`)
2. `index.html`, `about.html`, `README.md` 세 파일을 저장소에 업로드합니다.
3. 저장소 **Settings → Pages** 로 이동 → Source를 **`main` 브랜치 / `/ (root)`** 로 설정하고 Save.
4. 1~2분 후 `https://<your-username>.github.io` 에서 사이트가 열립니다.
